# Orders App Landing

Небольшой приветственный лендинг на чистом HTML/CSS/JS с черновым модулем учета заказов.

## Что внутри

- Приветственный лендинг: hero, преимущества, блок о проекте, CTA.
- Черновой модуль учета заказов (скрыт по умолчанию, открывается по кнопке).
- Хранение заказов в `localStorage` браузера.

## Быстрый запуск

Проект не требует сборки:

1. Откройте файл `index.html` в браузере.
2. Или запустите любой статический сервер в папке проекта.

## Структура

- `index.html` — разметка лендинга и teaser-секция модуля.
- `styles.css` — стили лендинга и интерфейса заказов.
- `app.js` — интерактив лендинга и логика модуля заказов.
